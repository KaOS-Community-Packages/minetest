# minetest
An InfiniMiner/Minecraft inspired game.

### Dependencies
Install them in order

````bash
kcp -i jemalloc
kcp -i redis
kcp -i hiredis
kcp -i gperftools
kcp -i leveldb
kcp -i snappy
kcp -i luajit
kcp -i irrlicht
````

### Game

````bash
kcp -i minetest
````
